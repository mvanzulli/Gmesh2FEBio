# Gmesh2FEBio
An under construction API to use FEBio with Gmesh meshes 

1. Create virtual environment and install necessary modules with `pip`:

    Create venv:

    ```
    python3 -m venv .venv
    ```

    Activate it:

    ```
    . .venv/bin/activate
    ```

    ```
    pip install -r requirements-dev.txt
    ```
